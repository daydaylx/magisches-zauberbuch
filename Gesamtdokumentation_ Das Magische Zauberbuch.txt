Zusammenfassung der Datenbankstruktur für "Das Magische Zauberbuch"
Dieses Dokument beschreibt die Struktur der PostgreSQL-Datenbank, die über Supabase für das PWA-Textadventure "Das Magische Zauberbuch" verwaltet wird.
Hilfsfunktionen
* public.handle_updated_at() : Eine Trigger-Funktion, die automatisch das Feld updated_at jeder Tabelle aktualisiert, wenn eine Zeile geändert wird.
Tabellenübersicht
1. public.game_settings
* Zweck: Speichert globale Einstellungen für das Spiel. Es ist so konzipiert, dass es idealerweise nur eine Zeile für die primären Einstellungen gibt, identifiziert durch setting_key.
* Spalten:
   * id (UUID, PK, DEFAULT gen_random_uuid()): Eindeutiger Identifikator.
   * created_at (TIMESTAMPTZ, NOT NULL, DEFAULT now()): Zeitstempel der Erstellung.
   * updated_at (TIMESTAMPTZ, NOT NULL, DEFAULT now()): Zeitstempel der letzten Aktualisierung.
   * setting_key (TEXT, UNIQUE, NOT NULL, DEFAULT 'primary_settings'): Ein Schlüssel, um die Einstellungszeile eindeutig zu identifizieren (z.B. "primary_settings").
   * main_plot_outline_for_ai (TEXT): Grobe Handlungsübersicht für die KI.
   * global_tone_style_for_ai (TEXT): Globaler Ton und Stil für KI-generierte Inhalte.
   * default_player_char_id (UUID, FK REFERENCES characters(id) ON DELETE SET NULL): ID des Standard-Spielercharakters (optional).
   * default_start_scene_slug (TEXT): Slug der Standard-Startszene (optional).
   * ai_model_preference (TEXT, DEFAULT 'openai/gpt-4o'): Bevorzugtes KI-Modell.
   * global_ai_params (JSONB, DEFAULT '{}'): Globale Parameter für die KI (z.B. Temperatur, max_tokens).
* Trigger: on_game_settings_updated (ruft handle_updated_at auf).
* RLS Policies:
   * Allow public read access to game settings: Erlaubt Lesezugriff für alle.
   * Allow admin full access to game settings: Erlaubt vollen Zugriff für die service_role (oder eine spezifische Admin-Rolle).
2. public.characters
* Zweck: Definiert Spielercharaktere und NPCs.
* Spalten:
   * id (UUID, PK, DEFAULT gen_random_uuid()): Eindeutiger Identifikator.
   * created_at (TIMESTAMPTZ, NOT NULL, DEFAULT now()): Zeitstempel der Erstellung.
   * updated_at (TIMESTAMPTZ, NOT NULL, DEFAULT now()): Zeitstempel der letzten Aktualisierung.
   * character_slug (TEXT, UNIQUE, NOT NULL): Eindeutiger Kurzname/Slug für den Charakter.
   * name (TEXT, NOT NULL): Anzeigename des Charakters.
   * is_player_character (BOOLEAN, NOT NULL, DEFAULT FALSE): Gibt an, ob dies ein Spielercharakter ist.
   * description_for_ai (TEXT): Beschreibung des Charakters für den KI-Kontext.
   * initial_stats (JSONB, DEFAULT '{}'): Initiale Charakterstatistiken.
   * initial_inventory (JSONB, DEFAULT '[]'): Initiales Inventar des Charakters.
   * initial_flags (JSONB, DEFAULT '{}'): Initiale Flags, die für diesen Charakter gesetzt sind.
   * avatar_url (TEXT): URL zu einem Avatarbild.
* Indizes: idx_character_slug, idx_is_player_character.
* Trigger: on_characters_updated (ruft handle_updated_at auf).
* RLS Policies:
   * Allow public read access to characters: Erlaubt Lesezugriff für alle.
   * Allow admin full access to characters: Erlaubt vollen Zugriff für die service_role.
3. public.scenes
* Zweck: Speichert die einzelnen Szenen des Spiels.
* Spalten:
   * id (UUID, PK, DEFAULT gen_random_uuid()): Eindeutiger Identifikator.
   * created_at (TIMESTAMPTZ, NOT NULL, DEFAULT now()): Zeitstempel der Erstellung.
   * updated_at (TIMESTAMPTZ, NOT NULL, DEFAULT now()): Zeitstempel der letzten Aktualisierung.
   * scene_slug (TEXT, UNIQUE, NOT NULL): Eindeutiger Kurzname/Slug für die Szene.
   * title_for_admin (TEXT): Titel der Szene für das Admin-Panel.
   * text_for_player (TEXT): Hauptbeschreibungstext der Szene für den Spieler.
   * image_url (TEXT): Optionale URL zu einem Szenenbild.
   * music_url (TEXT): Optionale URL zu Hintergrundmusik.
   * context_description_for_ai (TEXT): Kontextbeschreibung für die KI, wenn von dieser Szene aus generiert wird.
   * initial_prompt_for_ai (TEXT): Spezifischer Prompt für die KI, falls diese Szene ein Startpunkt für die Generierung ist.
   * is_ai_generated (BOOLEAN, NOT NULL, DEFAULT FALSE): Gibt an, ob der Szeneninhalt KI-generiert wurde.
   * tags (TEXT[] DEFAULT '{}'): Array für Tags (Hinzugefügt basierend auf späterer SQL-Migration)
* Indizes: idx_scene_slug, idx_scene_tags (GIN-Index für Array-Suche, hinzugefügt basierend auf späterer SQL-Migration).
* Trigger: on_scenes_updated (ruft handle_updated_at auf).
* RLS Policies:
   * Allow public read access to scenes: Erlaubt Lesezugriff für alle.
   * Allow admin full access to scenes: Erlaubt vollen Zugriff für die service_role.
4. public.choices
* Zweck: Speichert die Entscheidungsoptionen, die einem Spieler in einer bestimmten Szene zur Verfügung stehen.
* Spalten:
   * id (UUID, PK, DEFAULT gen_random_uuid()): Eindeutiger Identifikator.
   * created_at (TIMESTAMPTZ, NOT NULL, DEFAULT now()): Zeitstempel der Erstellung.
   * updated_at (TIMESTAMPTZ, NOT NULL, DEFAULT now()): Zeitstempel der letzten Aktualisierung.
   * scene_id (UUID, NOT NULL, FK REFERENCES scenes(id) ON DELETE CASCADE): ID der Szene, zu der diese Option gehört. Wird die Szene gelöscht, werden auch die Optionen gelöscht.
   * order_in_scene (INTEGER, NOT NULL, DEFAULT 0): Reihenfolge der Anzeige der Optionen in der Szene.
   * player_facing_text (TEXT, NOT NULL): Text der Option, der dem Spieler angezeigt wird.
   * ai_prompt_modifier_for_next (TEXT): Zusätzlicher Text, der dem KI-Prompt hinzugefügt wird, wenn diese Option gewählt wird.
   * leads_to_specific_scene_slug (TEXT): Slug einer vordefinierten Szene, zu der diese Option führt (optional).
   * einfluss_level (TEXT, CHECK (einfluss_level IN ('niedrig', 'mittel', 'hoch')), DEFAULT 'niedrig'): Einflusslevel der Entscheidung.
   * flags_to_set (JSONB, DEFAULT '{}'): Flags, die gesetzt/aktualisiert werden, wenn diese Option gewählt wird.
   * required_flags (JSONB, DEFAULT '{}'): Flags, die erforderlich sind, damit diese Option sichtbar/verfügbar ist.
* Indizes: idx_choices_scene_id, idx_choices_order.
* Trigger: on_choices_updated (ruft handle_updated_at auf).
* RLS Policies:
   * Allow public read access to choices: Erlaubt Lesezugriff für alle.
   * Allow admin full access to choices: Erlaubt vollen Zugriff für die service_role.
5. public.flags_definitions
* Zweck: Definiert die im Spiel verfügbaren Flags/Variablen, deren Typen und Standardwerte zur Referenz im Admin-Panel.
* Spalten:
   * id (UUID, PK, DEFAULT gen_random_uuid()): Eindeutiger Identifikator.
   * created_at (TIMESTAMPTZ, NOT NULL, DEFAULT now()): Zeitstempel der Erstellung.
   * updated_at (TIMESTAMPTZ, NOT NULL, DEFAULT now()): Zeitstempel der letzten Aktualisierung.
   * flag_name (TEXT, UNIQUE, NOT NULL): Eindeutiger Name des Flags (z.B. "hatMagischenSchluessel", "spielerGesundheit").
   * flag_type (TEXT, NOT NULL, CHECK (flag_type IN ('BOOLEAN', 'NUMBER', 'STRING'))): Datentyp des Flags.
   * default_value (TEXT): Standardwert des Flags (als Text gespeichert, wird basierend auf flag_type interpretiert).
   * description (TEXT): Beschreibung des Flags für das Admin-Panel.
* Indizes: idx_flag_name.
* Trigger: on_flags_definitions_updated (ruft handle_updated_at auf).
* RLS Policies:
   * Allow public read access to flag definitions: Erlaubt Lesezugriff für alle.
   * Allow admin full access to flag definitions: Erlaubt vollen Zugriff für die service_role.
6. public.player_sessions
* Zweck: Speichert den Zustand der aktuellen Spielsitzung eines Spielers.
* Spalten:
   * id (UUID, PK, DEFAULT gen_random_uuid()): Eindeutiger Identifikator der Spielsitzung.
   * created_at (TIMESTAMPTZ, NOT NULL, DEFAULT now()): Zeitstempel der Erstellung.
   * updated_at (TIMESTAMPTZ, NOT NULL, DEFAULT now()): Zeitstempel der letzten Aktualisierung.
   * player_id (UUID, FK REFERENCES auth.users(id) ON DELETE SET NULL): ID des Supabase-Auth-Benutzers (optional, falls anonyme Sessions erlaubt sind).
   * current_scene_id (UUID, FK REFERENCES scenes(id) ON DELETE SET NULL): ID der aktuellen Szene, in der sich der Spieler befindet.
   * player_flags (JSONB, DEFAULT '{}'): Aktueller Zustand der Flags des Spielers.
   * player_stats (JSONB, DEFAULT '{}'): Aktuelle Statistiken des Spielers.
   * player_inventory (JSONB, DEFAULT '[]'): Aktuelles Inventar des Spielers.
   * last_played_at (TIMESTAMPTZ, DEFAULT now()): Zeitstempel der letzten Interaktion.
* Indizes: idx_player_sessions_player_id, idx_player_sessions_current_scene_id.
* Trigger: on_player_sessions_updated (ruft handle_updated_at auf).
* RLS Policies:
   * Players can manage their own sessions: Erlaubt Spielern vollen Zugriff (Lesen, Schreiben, Löschen) nur auf ihre eigenen Sessions (basierend auf auth.uid() = player_id).
   * Service role full access to player sessions: Erlaubt der service_role vollen Zugriff auf alle Sessions.
Beziehungen zwischen den Tabellen
* game_settings kann auf einen default_player_char_id aus characters verweisen.
* scenes ist die zentrale Tabelle für Spielinhalte.
* choices gehört immer zu genau einer scene (1:N Beziehung, scenes zu choices).
* player_sessions verweist auf die current_scene_id aus scenes und optional auf einen player_id aus auth.users.
* flags_definitions ist eine Referenztabelle, die im Spiel-Logik und im Admin-Panel verwendet wird, um Flags zu definieren und zu verwalten. Die tatsächlichen Werte der Flags für einen Spieler werden in player_sessions.player_flags oder characters.initial_flags gespeichert.
Diese Struktur bietet eine flexible Basis für dein textbasiertes Abenteuerspiel und das zugehörige Admin-Panel.

# Das Magische Zauberbuch - PWA Textadventure

Willkommen zum PWA Textadventure "Das Magische Zauberbuch"! Dieses Projekt umfasst ein React-Frontend (erstellt mit Vite und TypeScript) für das Spiel und ein Admin-Panel zur Verwaltung der Spielinhalte, sowie ein Supabase-Backend für die Datenspeicherung und serverseitige Logik (Edge Functions).

## Features (Geplant & Teilimplementiert)

* **Spieler-Frontend**:
   * PWA für Offline-Fähigkeit.
   * KI-gesteuerte Story-Generierung über OpenRouter (via Supabase Edge Function).
   * Dynamische Szenen und Entscheidungen.
* **Admin-Panel**:
   * Login-System.
   * CRUD-Operationen für Szenen, Charaktere, Flags und Spieleinstellungen.
   * Vorschau-Modus für Spielszenen.
* **Backend (Supabase)**:
   * PostgreSQL-Datenbank.
   * Benutzerauthentifizierung.
   * Edge Functions für serverseitige Logik (z.B. KI-Anbindung, Spieler-Session-Management).

## Setup & Installation
1.  **Repository klonen**:
   ```bash
   git clone <repository-url>
   cd zauberbuch-pwa
   ```
2.  **Abhängigkeiten installieren**:
   ```bash
   npm install
   # oder
   # yarn install
   ```
3.  **Supabase Setup**:
   * Erstelle ein Projekt auf [Supabase](https://supabase.com).
   * Navigiere zu "Project Settings" > "API". Kopiere die **Project URL** und den **anon public Key**.
   * Erstelle die Datenbanktabellen. Du kannst die SQL-Skripte aus `supabase/migrations/` im Supabase SQL Editor ausführen.
   * Konfiguriere Row-Level Security (RLS) Policies für deine Tabellen.
   * Hinterlege deinen OpenRouter API Key als Secret in den Supabase Edge Function Einstellungen (z.B. `OPENROUTER_API_KEY`).
   * Hinterlege deinen Supabase Service Role Key als Secret (z.B. `SUPABASE_SERVICE_ROLE_KEY`).
4.  **Umgebungsvariablen einrichten**:
   * Kopiere `.env.example` zu `.env.local`.
       ```bash
       cp .env.example .env.local
       ```
   * Trage deine Supabase URL und den Anon Key in `.env.local` ein:
       ```env
       VITE_SUPABASE_URL=DEINE_SUPABASE_URL
       VITE_SUPABASE_ANON_KEY=DEIN_SUPABASE_ANON_KEY
       ```
       *(Für Vite müssen Umgebungsvariablen, die im Client-Code verfügbar sein sollen, mit `VITE_` beginnen.)*
5.  **Supabase CLI (Optional, aber empfohlen für Functions & Migrationen)**:
   * [Installiere die Supabase CLI](https://supabase.com/docs/guides/cli).
   * Verbinde die CLI mit deinem Projekt:
       ```bash
       supabase login
       supabase link --project-ref <DEIN_PROJEKT_REF>
       # Ggf. supabase start, falls lokal entwickelt wird
       ```
   * Deploye die Edge Functions:
       ```bash
       supabase functions deploy generate-next-scene
       supabase functions deploy create-player-session
       # Stelle sicher, dass die Secrets (OPENROUTER_API_KEY, SUPABASE_SERVICE_ROLE_KEY) im Supabase Dashboard gesetzt sind.
       ```
6.  **Entwicklungsserver starten**:
   ```bash
   npm run dev
   # oder
   # yarn dev
   ```
   Die Anwendung sollte unter `http://localhost:5173` (oder einem ähnlichen Port) erreichbar sein.

## Verfügbare Skripte
* `npm run dev`: Startet den Vite Entwicklungsserver.
* `npm run build`: Erstellt einen optimierten Build für die Produktion im `dist` Ordner.
* `npm run preview`: Startet einen lokalen Server, um den Produktionsbuild zu testen.
* `npm run lint`: Führt ESLint aus, um Code-Qualität zu prüfen.

## Admin Panel
Das Admin Panel ist unter `/admin` zugänglich. Du musst dich mit den konfigurierten Admin-Benutzerdaten anmelden.

## Technologie-Stack
* **Frontend**: React, TypeScript, Vite, Material UI
* **Backend**: Supabase (Auth, PostgreSQL, Edge Functions)
* **KI-Anbindung**: OpenRouter API

## Setup & Installation

1.  **Repository klonen**:
   ```bash
   git clone <repository-url>
   cd zauberbuch-pwa
   ```
2.  **Abhängigkeiten installieren**:
   ```bash
   npm install
   # oder
   # yarn install
   ```
3.  **Supabase Setup**:
   * Erstelle ein Projekt auf [Supabase](https://supabase.com).
   * Navigiere zu "Project Settings" > "API". Kopiere die **Project URL** und den **anon public Key**.
   * Erstelle die Datenbanktabellen. Du kannst die SQL-Skripte aus `supabase/migrations/` im Supabase SQL Editor ausführen.
   * Konfiguriere Row-Level Security (RLS) Policies für deine Tabellen.
   * Hinterlege deinen OpenRouter API Key als Secret in den Supabase Edge Function Einstellungen (z.B. `OPENROUTER_API_KEY`).
   * Hinterlege deinen Supabase Service Role Key als Secret (z.B. `SUPABASE_SERVICE_ROLE_KEY`).
4.  **Umgebungsvariablen einrichten**:
   * Kopiere `.env.example` zu `.env.local`.
       ```bash
       cp .env.example .env.local
       ```
   * Trage deine Supabase URL und den Anon Key in `.env.local` ein:
       ```env
       VITE_SUPABASE_URL=DEINE_SUPABASE_URL
       VITE_SUPABASE_ANON_KEY=DEIN_SUPABASE_ANON_KEY
       ```
       *(Für Vite müssen Umgebungsvariablen, die im Client-Code verfügbar sein sollen, mit `VITE_` beginnen.)*
5.  **Supabase CLI (Optional, aber empfohlen für Functions & Migrationen)**:
   * [Installiere die Supabase CLI](https://supabase.com/docs/guides/cli).
   * Verbinde die CLI mit deinem Projekt:
       ```bash
       supabase login
       supabase link --project-ref <DEIN_PROJEKT_REF>
       # Ggf. supabase start, falls lokal entwickelt wird
       ```
   * Deploye die Edge Functions:
       ```bash
       supabase functions deploy generate-next-scene
       supabase functions deploy create-player-session
       # Stelle sicher, dass die Secrets (OPENROUTER_API_KEY, SUPABASE_SERVICE_ROLE_KEY) im Supabase Dashboard gesetzt sind.
       ```
6.  **Entwicklungsserver starten**:
   ```bash
   npm run dev
   # oder
   # yarn dev
   ```
   Die Anwendung sollte unter `http://localhost:5173` (oder einem ähnlichen Port) erreichbar sein.

RETURNS TRIGGER AS $$
BEGIN
   NEW.updated_at = now();
   RETURN NEW;
END;

-- RLS Policies Beispiele:
-- Diese sind grundlegende Beispiele. Du musst sie basierend auf deiner genauen Auth-Logik verfeinern.

-- Für game_settings:
CREATE POLICY "Allow public read access to game settings" ON public.game_settings
 FOR SELECT USING (true);
CREATE POLICY "Allow admin full access to game settings" ON public.game_settings
   FOR ALL
   USING (auth.role() = 'service_role') -- Oder prüfe auf eine spezifische Admin-Rolle
   WITH CHECK (auth.role() = 'service_role');

-- Für characters, scenes, choices, flags_definitions (öffentlicher Lesezugriff, Admin Vollzugriff):
CREATE POLICY "Allow public read access to characters" ON public.characters FOR SELECT USING (true);
CREATE POLICY "Allow admin full access to characters" ON public.characters FOR ALL USING (auth.role() = 'service_role') WITH CHECK (auth.role() = 'service_role');

CREATE POLICY "Allow public read access to scenes" ON public.scenes FOR SELECT USING (true);
CREATE POLICY "Allow admin full access to scenes" ON public.scenes FOR ALL USING (auth.role() = 'service_role') WITH CHECK (auth.role() = 'service_role');

CREATE POLICY "Allow public read access to choices" ON public.choices FOR SELECT USING (true);
CREATE POLICY "Allow admin full access to choices" ON public.choices FOR ALL USING (auth.role() = 'service_role') WITH CHECK (auth.role() = 'service_role');

CREATE POLICY "Allow public read access to flag definitions" ON public.flags_definitions FOR SELECT USING (true);
CREATE POLICY "Allow admin full access to flag definitions" ON public.flags_definitions FOR ALL USING (auth.role() = 'service_role') WITH CHECK (auth.role() = 'service_role');


-- Für player_sessions:
CREATE POLICY "Players can manage their own sessions" ON public.player_sessions
 FOR ALL
 USING (auth.uid() = player_id)
 WITH CHECK (auth.uid() = player_id);

CREATE POLICY "Service role full access to player sessions" ON public.player_sessions
 FOR ALL
 USING (auth.role() = 'service_role')
 WITH CHECK (auth.role() = 'service_role');

 image_prompt?: string; // Optionaler Prompt für ein Bildgenerierungsmodell
 music_url?: string;    // Optionale URL für Hintergrundmusik
 error?: string;        // Fehlerfeld für die KI-Antwort
}

// Funktion zum Aufrufen der OpenRouter API
async function callOpenRouter(
 prompt: string,
 apiKey: string,
 model: string = 'openai/gpt-4o', // Standardmodell, kann über game_settings überschrieben werden
 globalAiParams: Record<string, any> = {} // Globale Parameter aus game_settings
): Promise<AiGeneratedContent> {
 console.log(`Calling OpenRouter with model: ${model}`);
 const body = {
   model: model,
   messages: [
     {
       role: "system",
       content: `Du bist ein Storytelling-Assistent für ein textbasiertes Abenteuerspiel. Generiere eine fesselnde, kurze nächste Szene (max. 3-4 Sätze) und 2-4 passende Multiple-Choice-Entscheidungen für den Spieler. Antworte IMMER im JSON-Format: {
 "new_scene_title": "Titel der neuen Szene (kurz)",
 "new_scene_description": "Beschreibung der neuen Szene.",
 "new_choices": [
   { "text": "Option 1 Text" },
   { "text": "Option 2 Text" }
 ],
 "image_prompt": "Optionaler Prompt für ein Bild zu dieser Szene (z.B. 'mystischer Wald bei Nacht')",
 "music_url": "Optionale URL zu passender Hintergrundmusik (z.B. Creative Commons)"
}
Achte darauf, dass die Entscheidungen offen genug sind, um die Geschichte interessant weiterzuführen. Vermeide es, das Spiel zu beenden, es sei denn, der Prompt fordert explizit ein Ende. Die 'new_choices' sollten nur den 'text' enthalten. Keine weiteren Felder wie 'sets_flag_id'.`
     },
     { role: "user", content: prompt }
   ],
   temperature: globalAiParams.temperature || 0.7, // Nutze globale oder Standard-Temperatur
   max_tokens: globalAiParams.max_tokens || 500,   // Nutze globale oder Standard-max_tokens
   ...globalAiParams // Füge weitere globale Parameter hinzu (z.B. top_p etc.)
 };

 const response = await fetch("https://openrouter.ai/api/v1/chat/completions", {
   method: "POST",
   headers: {
     "Authorization": `Bearer ${apiKey}`,
     "Content-Type": "application/json",
     "HTTP-Referer": Deno.env.get("SUPABASE_URL") ?? "http://localhost", // Wichtig für OpenRouter
     "X-Title": "Zauberbuch PWA" // Wichtig für OpenRouter
   },
   body: JSON.stringify(body),
 });

 if (!response.ok) {
   const errTxt = await response.text();
   console.error("OpenRouter API Error:", response.status, errTxt);
   return {
     new_scene_title: "Fehler bei der Szenengenerierung",
     new_scene_description: `Ein Fehler ist beim Kontaktieren des Orakels aufgetreten (Fehlercode: ${response.status}). Versuche es später erneut oder wähle eine andere Option.`,
     new_choices: [{ text: "Zurück zur vorherigen Entscheidung (simuliert)" }], // Simuliert, da wir den vorherigen Zustand hier nicht haben
     error: `OpenRouter API Error: ${response.status} - ${errTxt}`
   };
 }

 const data = await response.json();

 try {
   // Die KI soll direkt ein JSON-Objekt im 'content'-Feld zurückgeben.
   const messageContent = data.choices[0].message.content;
   let parsedResponse: AiGeneratedContent;

   if (typeof messageContent === 'string') {
     // Versuche, den String als JSON zu parsen
     parsedResponse = JSON.parse(messageContent);
   } else if (typeof messageContent === 'object' && messageContent !== null) {
     // Wenn es bereits ein Objekt ist (manche Modelle tun das)
     parsedResponse = messageContent as AiGeneratedContent;
   } else {
     throw new Error("Unerwartetes Format des message.content Feldes.");
   }

   // Validierung der minimal erforderlichen Felder
   if (!parsedResponse.new_scene_description || !parsedResponse.new_choices || !Array.isArray(parsedResponse.new_choices) || parsedResponse.new_choices.length === 0) {
     console.error("Unexpected AI response format after parsing:", parsedResponse);
     throw new Error("AI response format error: new_scene_description or new_choices are missing or invalid.");
   }
   // Stelle sicher, dass choices das richtige Format haben
   parsedResponse.new_choices = parsedResponse.new_choices.map(choice => ({ text: choice.text }));

   return parsedResponse;
 } catch (e: any) {
   console.error("Error parsing AI response:", e, data?.choices?.[0]?.message?.content);
   return {
     new_scene_title: "Fehler beim Verstehen des Orakels",
     new_scene_description: "Das Orakel sprach in Rätseln, die nicht gedeutet werden konnten. Die Verbindung ist gestört.",
     new_choices: [{ text: "Erneut versuchen (simuliert)" }],
     error: `Error parsing AI response: ${e.message}`
   };
 }
}


// Haupt-Handler für die Edge Function
serve(async (req: Request) => {
 // Handle CORS preflight requests
 if (req.method === 'OPTIONS') {
   return new Response('ok', { headers: corsHeaders });
 }

 try {
   const {
     player_session_id, // UUID der aktuellen Spielsitzung
     chosen_option_id   // UUID der vom Spieler gewählten Option
   } = await req.json();

   if (!player_session_id || !chosen_option_id) {
     return new Response(JSON.stringify({ error: "player_session_id and chosen_option_id are required." }), {
       headers: { ...corsHeaders, 'Content-Type': 'application/json' },
       status: 400
     });
   }

   // Supabase Client initialisieren, um auf die Datenbank zuzugreifen
   // Wichtig: Verwende den SERVICE_ROLE_KEY für Operationen, die Admin-Rechte erfordern oder RLS umgehen sollen.
   const supabaseClient = createClient(
     Deno.env.get('SUPABASE_URL')!,
     Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')! // SERVICE_ROLE_KEY für Backend-Operationen
   );

   // 1. Lade aktuelle Player Session und die zugehörige Szene und gewählte Option
   const { data: sessionData, error: sessionError } = await supabaseClient
     .from('player_sessions')
     .select(`
       id,
       player_flags,
       player_stats,
       player_inventory,
       current_scene:scenes (
         id,
         scene_slug,
         text_for_player,
         context_description_for_ai,
         initial_prompt_for_ai
       )
     `)
     .eq('id', player_session_id)
     .single();

   if (sessionError || !sessionData || !sessionData.current_scene) {
     console.error('Error fetching player session or current scene:', sessionError);
     return new Response(JSON.stringify({ error: 'Player session or current scene not found.' }), {
       headers: { ...corsHeaders, 'Content-Type': 'application/json' },
       status: 404
     });
   }

   const { data: choiceData, error: choiceError } = await supabaseClient
     .from('choices')
     .select('*')
     .eq('id', chosen_option_id)
     .eq('scene_id', sessionData.current_scene.id) // Sicherstellen, dass die Option zur aktuellen Szene gehört
     .single();

   if (choiceError || !choiceData) {
     console.error('Error fetching chosen option:', choiceError);
     return new Response(JSON.stringify({ error: 'Chosen option not found or does not belong to the current scene.' }), {
       headers: { ...corsHeaders, 'Content-Type': 'application/json' },
       status: 404
     });
   }

   // 2. Flags basierend auf der Wahl aktualisieren (aus choiceData.flags_to_set)
   let updatedPlayerFlags = { ...(sessionData.player_flags || {}) };
   if (choiceData.flags_to_set) {
     for (const key in choiceData.flags_to_set) {
       const value = choiceData.flags_to_set[key];
       if (typeof value === 'string' && value.startsWith('+') && !isNaN(Number(value.substring(1)))) {
         updatedPlayerFlags[key] = (updatedPlayerFlags[key] || 0) + Number(value.substring(1));
       } else if (typeof value === 'string' && value.startsWith('-') && !isNaN(Number(value.substring(1)))) {
         updatedPlayerFlags[key] = (updatedPlayerFlags[key] || 0) - Number(value.substring(1));
       } else {
         updatedPlayerFlags[key] = value;
       }
     }
   }

   let nextSceneRecord;
   let nextChoicesData = [];

   // 3. Bestimme die nächste Szene
   if (choiceData.leads_to_specific_scene_slug) {
     // Option führt zu einer vordefinierten Szene
     const { data: specificScene, error: specificSceneError } = await supabaseClient
       .from('scenes')
       .select('*, choices(*, order_in_scene)') // Lade auch die Choices für die spezifische Szene
       .eq('scene_slug', choiceData.leads_to_specific_scene_slug)
       .single();

     if (specificSceneError || !specificScene) {
       console.error(`Error fetching specific next scene by slug '${choiceData.leads_to_specific_scene_slug}':`, specificSceneError);
       return new Response(JSON.stringify({ error: `Specific next scene '${choiceData.leads_to_specific_scene_slug}' not found.` }), {
         headers: { ...corsHeaders, 'Content-Type': 'application/json' },
         status: 404
       });
     }
     nextSceneRecord = specificScene;
     nextChoicesData = (specificScene.choices || [])
       .sort((a: any,b: any) => (a.order_in_scene ?? 0) - (b.order_in_scene ?? 0))
       .map((c: any) => ({ id: c.id, text: c.player_facing_text }));

   } else {
     // KI generiert die nächste Szene
     const apiKey = Deno.env.get('OPENROUTER_API_KEY');
     if (!apiKey) {
       console.error('OPENROUTER_API_KEY not set for AI generation.');
       return new Response(JSON.stringify({ error: 'API Key for AI generation is missing on server-side.' }), {
         headers: { ...corsHeaders, 'Content-Type': 'application/json' },
         status: 500
       });
     }

     const { data: gameSettings, error: gsError } = await supabaseClient
       .from('game_settings')
       .select('ai_model_preference, global_ai_params, main_plot_outline_for_ai, global_tone_style_for_ai')
       .eq('setting_key', 'primary_settings')
       .single();

     if (gsError) console.warn("Could not load game settings for AI parameters.", gsError.message);

     let prompt = `Kontext: ${sessionData.current_scene.context_description_for_ai || sessionData.current_scene.text_for_player}\n`;
     if (sessionData.current_scene.initial_prompt_for_ai) {
       prompt += `Ursprünglicher Prompt für diese Szene: ${sessionData.current_scene.initial_prompt_for_ai}\n`;
     }
     prompt += `Spieler hat gewählt: "${choiceData.player_facing_text}".\n`;
     if (choiceData.ai_prompt_modifier_for_next) {
       prompt += `Zusätzlicher Kontext für diese Wahl: "${choiceData.ai_prompt_modifier_for_next}".\n`;
     }
     prompt += `Aktuelle Spieler-Flags: ${JSON.stringify(updatedPlayerFlags)}.\n`;
     if (gameSettings?.main_plot_outline_for_ai) {
       prompt += `Übergeordneter Plot: ${gameSettings.main_plot_outline_for_ai}\n`;
     }
     if (gameSettings?.global_tone_style_for_ai) {
       prompt += `Gewünschter Ton: ${gameSettings.global_tone_style_for_ai}\n`;
     }
     prompt += `Generiere jetzt den Titel der nächsten Szene, den Text für die nächste Szene und 2-4 prägnante Entscheidungsoptionen für den Spieler im JSON-Format.`;

     const aiResponse = await callOpenRouter(
       prompt,
       apiKey,
       gameSettings?.ai_model_preference || undefined,
       gameSettings?.global_ai_params || undefined
     );

     if (aiResponse.error) {
       return new Response(JSON.stringify(aiResponse), { // aiResponse enthält bereits Fehlerdetails
         headers: { ...corsHeaders, 'Content-Type': 'application/json' },
         status: 500 // Oder einen spezifischeren Fehlercode, falls von aiResponse bereitgestellt
       });
     }

     // Erstelle einen einzigartigen Slug für die KI-generierte Szene
     const aiSceneSlug = `ki-${sessionData.current_scene.scene_slug}-${Date.now()}`;

     // Speichere die KI-generierte Szene in der 'scenes' Tabelle
     const { data: tempScene, error: tempSceneError } = await supabaseClient
       .from('scenes')
       .insert({
         scene_slug: aiSceneSlug,
         title_for_admin: aiResponse.new_scene_title || `KI Szene nach: ${sessionData.current_scene.scene_slug}`,
         text_for_player: aiResponse.new_scene_description,
         image_url: aiResponse.image_prompt ? `https://image.pollinations.ai/prompt/${encodeURIComponent(aiResponse.image_prompt)}` : null, // Beispiel für Bildgenerierung
         music_url: aiResponse.music_url, // Falls von KI bereitgestellt
         context_description_for_ai: `KI-generiert basierend auf Wahl: "${choiceData.player_facing_text}" in Szene "${sessionData.current_scene.scene_slug}". Spieler-Prompt: ${aiResponse.new_scene_description.substring(0,100)}...`,
         initial_prompt_for_ai: "Fortsetzung der KI-generierten Geschichte.",
         is_ai_generated: true,
         // tags könnten hier auch basierend auf dem Inhalt gesetzt werden, falls gewünscht
       })
       .select()
       .single();

     if (tempSceneError || !tempScene) {
       console.error('Error inserting AI generated scene:', tempSceneError);
       return new Response(JSON.stringify({ error: 'Could not save AI generated scene.' }), {
         headers: { ...corsHeaders, 'Content-Type': 'application/json' },
         status: 500
       });
     }
     nextSceneRecord = tempScene;

     // Speichere die KI-generierten Choices in der 'choices' Tabelle
     const tempChoicesToSave = aiResponse.new_choices.map((choice, index) => ({
       scene_id: tempScene.id,
       player_facing_text: choice.text,
       order_in_scene: index,
       // Weitere Felder wie leads_to_specific_scene_slug, flags_to_set etc. könnten hier von der KI kommen,
       // aber im aktuellen System-Prompt sind sie nicht vorgesehen und werden daher nicht gesetzt.
     }));

     if (tempChoicesToSave.length > 0) {
       const { data: insertedTempChoices, error: tempChoicesError } = await supabaseClient
         .from('choices')
         .insert(tempChoicesToSave)
         .select();

       if (tempChoicesError) {
         console.error('Error inserting AI generated choices:', tempChoicesError);
         // Fahre trotzdem fort, aber die Szene hat dann keine Optionen
       } else {
         nextChoicesData = (insertedTempChoices || []).map((c: any) => ({ id: c.id, text: c.player_facing_text }));
       }
     }
   }

   // 4. Player Session in DB aktualisieren
   const { data: updatedSessionData, error: updateSessionError } = await supabaseClient
     .from('player_sessions')
     .update({
       current_scene_id: nextSceneRecord.id,
       player_flags: updatedPlayerFlags,
       last_played_at: new Date().toISOString(),
     })
     .eq('id', player_session_id)
     .select('id, player_flags, player_stats, player_inventory') // Wähle die aktualisierten Daten aus
     .single();

   if (updateSessionError) {
     console.error('Error updating player session:', updateSessionError);
     return new Response(JSON.stringify({ error: 'Could not update player session.' }), {
       headers: { ...corsHeaders, 'Content-Type': 'application/json' },
       status: 500
     });
   }

   // 5. Antwort an das Frontend senden
   return new Response(
     JSON.stringify({
       new_scene_id: nextSceneRecord.id,
       scene_slug: nextSceneRecord.scene_slug,
       scene_title: nextSceneRecord.title_for_admin, // oder text_for_player, je nach Bedarf
       scene_text: nextSceneRecord.text_for_player,
       image_url: nextSceneRecord.image_url,
       music_url: nextSceneRecord.music_url,
       choices: nextChoicesData,
       updated_player_state: { // Sende den aktualisierten Spielerstatus zurück
         flags: updatedSessionData?.player_flags || updatedPlayerFlags,
         stats: updatedSessionData?.player_stats || sessionData.player_stats, // Behalte alte Stats, falls nicht geändert
         inventory: updatedSessionData?.player_inventory || sessionData.player_inventory, // Behalte altes Inventar, falls nicht geändert
       },
     }),
     { headers: { ...corsHeaders, 'Content-Type': 'application/json' }, status: 200 }
   );

 } catch (error: any) {
   console.error('Critical error in generate-next-scene function:', error);
   return new Response(JSON.stringify({ error: error.message || 'An unexpected error occurred.' }), {
     headers: { ...corsHeaders, 'Content-Type': 'application/json' },
     status: 500
   });
 }
});

import { corsHeaders } from '../_shared/cors.ts';

interface CreateSessionRequest {
 player_id?: string; // Optional: Supabase auth.uid() des Spielers
 start_scene_slug?: string; // Optional: Spezifischer Start-Szenen-Slug
 character_id?: string; // Optional: Spezifische Charakter-ID für initiale Werte
}

serve(async (req: Request) => {
 // Handle CORS preflight requests
 if (req.method === 'OPTIONS') {
   return new Response('ok', { headers: corsHeaders });
 }

 try {
   const { player_id, start_scene_slug, character_id }: CreateSessionRequest = await req.json();

   const supabaseClient = createClient(
     Deno.env.get('SUPABASE_URL')!,
     Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')! // Service Role Key für Backend-Operationen
   );

   let effectiveStartSceneSlug = start_scene_slug;
   let effectiveCharacterId = character_id;
   let initialPlayerFlags = {};
   let initialPlayerStats = {};
   let initialPlayerInventory = [];

   // Wenn keine spezifischen Startparameter gegeben sind, lade Standardwerte aus game_settings
   if (!effectiveStartSceneSlug || !effectiveCharacterId) {
     const { data: gameSettings, error: settingsError } = await supabaseClient
       .from('game_settings')
       .select('default_start_scene_slug, default_player_char_id')
       .eq('setting_key', 'primary_settings')
       .single();

     if (settingsError || !gameSettings) {
       console.warn('Could not load default game settings or settings not found:', settingsError?.message);
       // Fahre fort, aber es könnten Fehler auftreten, wenn keine Startwerte vorhanden sind
     } else {
       if (!effectiveStartSceneSlug) {
         effectiveStartSceneSlug = gameSettings.default_start_scene_slug;
       }
       if (!effectiveCharacterId) {
         effectiveCharacterId = gameSettings.default_player_char_id;
       }
     }
   }

   if (!effectiveStartSceneSlug) {
     return new Response(JSON.stringify({ error: 'Start scene slug could not be determined (neither provided nor found in default settings).' }), {
       headers: { ...corsHeaders, 'Content-Type': 'application/json' }, status: 400
     });
   }

   // Lade die Startszene
   const { data: sceneData, error: sceneError } = await supabaseClient
     .from('scenes')
     .select('id, scene_slug, title_for_admin, text_for_player, image_url, music_url, choices(*, order_in_scene)') // Lade auch Choices für die Startszene
     .eq('scene_slug', effectiveStartSceneSlug)
     .single();

   if (sceneError || !sceneData) {
     console.error(`Error fetching start scene by slug '${effectiveStartSceneSlug}':`, sceneError);
     return new Response(JSON.stringify({ error: `Start scene '${effectiveStartSceneSlug}' not found.` }), {
       headers: { ...corsHeaders, 'Content-Type': 'application/json' }, status: 404
     });
   }

   // Lade den Spielercharakter für initiale Werte (Flags, Stats, Inventar)
   if (effectiveCharacterId) {
     const { data: charData, error: charError } = await supabaseClient
       .from('characters')
       .select('initial_flags, initial_stats, initial_inventory')
       .eq('id', effectiveCharacterId)
       .single();

     if (charError) {
       console.warn(`Could not load character data for id '${effectiveCharacterId}':`, charError.message);
       // Fahre fort, Spieler startet mit leeren Werten
     } else if (charData) {
       initialPlayerFlags = charData.initial_flags || {};
       initialPlayerStats = charData.initial_stats || {};
       initialPlayerInventory = charData.initial_inventory || [];
     }
   }

   // Erstelle die Player Session in der Datenbank
   const { data: sessionData, error: sessionInsertError } = await supabaseClient
     .from('player_sessions')
     .insert({
       player_id: player_id || null, // Kann null sein für anonyme Sessions
       current_scene_id: sceneData.id,
       player_flags: initialPlayerFlags,
       player_stats: initialPlayerStats,
       player_inventory: initialPlayerInventory,
       last_played_at: new Date().toISOString(),
     })
     .select() // Wähle die eingefügten Daten aus
     .single();

   if (sessionInsertError || !sessionData) {
     console.error('Error creating player session:', sessionInsertError);
     return new Response(JSON.stringify({ error: 'Could not create player session.' }), {
       headers: { ...corsHeaders, 'Content-Type': 'application/json' }, status: 500
     });
   }

   // Formatiere die Choices für die Antwort
   const initialChoices = (sceneData.choices || [])
       .sort((a: any,b: any) => (a.order_in_scene ?? 0) - (b.order_in_scene ?? 0))
       .map((c: any) => ({ id: c.id, text: c.player_facing_text }));

   // Antwort an das Frontend
   return new Response(
     JSON.stringify({
       player_session_id: sessionData.id,
       initial_scene: {
         id: sceneData.id,
         scene_slug: sceneData.scene_slug,
         title: sceneData.title_for_admin, // oder text_for_player als Titel?
         text: sceneData.text_for_player,
         image_url: sceneData.image_url,
         music_url: sceneData.music_url,
         choices: initialChoices,
       },
       player_state: { // Sende den initialen Spielerstatus
         flags: sessionData.player_flags,
         stats: sessionData.player_stats,
         inventory: sessionData.player_inventory,
       }
     }),
     { headers: { ...corsHeaders, 'Content-Type': 'application/json' }, status: 200 }
   );

 } catch (error: any) {
   console.error('Critical error in create-player-session function:', error);
   return new Response(JSON.stringify({ error: error.message || 'An unexpected error occurred.' }), {
     headers: { ...corsHeaders, 'Content-Type': 'application/json' }, status: 500
   });
 }

import React, { createContext, useContext, useState, useEffect, ReactNode } from 'react';
import { supabase } from '../services/supabaseClient';
import type { User, Session } from '../types/authTypes'; // Stellen Sie sicher, dass diese Typen existieren

interface AuthContextType {
 user: User | null;
 session: Session | null;
 isLoading: boolean;
 isAdmin: boolean; // Beispiel für eine Admin-Rolle
 signOut: () => Promise<void>;
}

const AuthContext = createContext<AuthContextType | undefined>(undefined);

interface AuthProviderProps {
 children: ReactNode;
}

export const AuthProvider: React.FC<AuthProviderProps> = ({ children }) => {
 const [user, setUser] = useState<User | null>(null);
 const [session, setSession] = useState<Session | null>(null);
 const [isLoading, setIsLoading] = useState(true);
 const [isAdmin, setIsAdmin] = useState(false);

 useEffect(() => {
   let mounted = true; // Verhindert State-Updates, wenn die Komponente unmounted ist

   const fetchSession = async () => {
     if (!mounted) return;
     setIsLoading(true);
     const { data: { session: currentSession }, error } = await supabase.auth.getSession();

     if (error) {
       console.error("Error fetching session:", error.message);
     }

     if (mounted) {
       setSession(currentSession);
       setUser(currentSession?.user ?? null);
       // Beispielhafte Admin-Prüfung (in einer echten App sicherer gestalten, z.B. über Custom Claims oder eine separate Admin-Tabelle)
       // Ersetzen Sie 'admin@example.com' durch Ihre tatsächliche Admin-Logik oder E-Mail.
       // Für den privaten Gebrauch ist dies vorerst ausreichend.
       setIsAdmin(currentSession?.user?.email === 'admin@example.com'); // TODO: Durch echte Admin-Logik ersetzen
       setIsLoading(false);
     }
   };

   fetchSession();

   const { data: authListener } = supabase.auth.onAuthStateChange(
     async (_event, newSession) => {
       if (mounted) {
         setSession(newSession);
         setUser(newSession?.user ?? null);
         // Aktualisieren Sie den Admin-Status auch hier
         setIsAdmin(newSession?.user?.email === 'admin@example.com'); // TODO: Durch echte Admin-Logik ersetzen
         if (isLoading) setIsLoading(false); // Setze isLoading nur beim ersten Mal oder wenn es noch true ist
       }
     }
   );

   return () => {
     mounted = false;
     authListener?.subscription?.unsubscribe();
   };
 }, []); // Keine Abhängigkeiten, da der Listener alles abdeckt und fetchSession nur initial läuft

 const signOut = async () => {
   setIsLoading(true); // Optional: Ladezustand während des Ausloggens anzeigen
   await supabase.auth.signOut();
   // User, Session und isAdmin werden durch den onAuthStateChange Listener automatisch zurückgesetzt.
 };

 const value = {
   user,
   session,
   isLoading,
   isAdmin,
   signOut,
 };

 return <AuthContext.Provider value={value}>{children}</AuthContext.Provider>;
};

export const useAuth = (): AuthContextType => {
 const context = useContext(AuthContext);
 if (context === undefined) {
   throw new Error('useAuth must be used within an AuthProvider');
 }
 return context;
};

import React, { createContext, useContext, ReactNode } from 'react';
import { useSnackbar, VariantType, SnackbarKey } from 'notistack';

interface NotificationContextType {
 showToast: (message: string, variant?: VariantType) => SnackbarKey;
}

const NotificationContext = createContext<NotificationContextType | undefined>(undefined);

interface NotificationProviderProps {
 children: ReactNode;
}

export const NotificationProvider: React.FC<NotificationProviderProps> = ({ children }) => {
 const { enqueueSnackbar } = useSnackbar();

 const showToast = (message: string, variant: VariantType = 'default'): SnackbarKey => {
   // Ruft enqueueSnackbar von notistack auf, um eine Benachrichtigung anzuzeigen.
   // message: Der Text, der in der Benachrichtigung angezeigt wird.
   // variant: Der Typ der Benachrichtigung ('default', 'error', 'success', 'warning', 'info').
   return enqueueSnackbar(message, { variant });
 };

 return (
   <NotificationContext.Provider value={{ showToast }}>
     {children}
   </NotificationContext.Provider>
 );
};

export const useNotification = (): NotificationContextType => {
 const context = useContext(NotificationContext);
 if (!context) {
   // Dieser Fehler wird ausgelöst, wenn useNotification außerhalb eines NotificationProvider
   // und des von notistack bereitgestellten SnackbarProvider verwendet wird.
   throw new Error('useNotification must be used within a NotificationProvider and SnackbarProvider');
 }
 return context;
};

// File: src/services/sceneService.ts
import { supabase } from './supabaseClient';
import type { SceneData, ChoiceData, SceneListEntry } from '../types/gameTypes';

// Interface für Paginierungs-, Sortier- und Suchparameter
export interface FetchScenesListParams {
 page?: number; // 0-basiert
 rowsPerPage?: number;
 orderBy?: keyof SceneListEntry | 'choices_count'; // Erlaube Sortierung nach 'choices_count'
 orderDirection?: 'asc' | 'desc';
 searchTerm?: string;
}

/**
* Ruft eine Liste von Szenen ab, mit Paginierung, Sortierung und Suchfunktion.
* @param params - Objekt mit Paginierungs-, Sortier- und Suchparametern.
* @returns Ein Objekt mit den Szenen und der Gesamtanzahl der Szenen.
*/
export const fetchScenesList = async (
 params: FetchScenesListParams = {}
): Promise<{ scenes: SceneListEntry[], totalCount: number }> => {
 const {
   page = 0,
   rowsPerPage = 10,
   orderBy = 'updated_at',
   orderDirection = 'desc',
   searchTerm = ''
 } = params;

 let query = supabase
   .from('scenes')
   .select('id, scene_slug, title_for_admin, updated_at, tags, choices(count)', { count: 'exact' });

 if (searchTerm) {
   query = query.or(`scene_slug.ilike.%${searchTerm}%,title_for_admin.ilike.%${searchTerm}%`);
 }

 if (orderBy && orderBy !== 'choices_count') {
        query = query.order(orderBy as string, { ascending: orderDirection === 'asc' });
 }
 if (orderBy !== 'id') { 
       query = query.order('id', { ascending: true });
 }

 const from = page * rowsPerPage;
 const to = from + rowsPerPage - 1;
 query = query.range(from, to);

 const { data, error, count } = await query;

 if (error) {
   console.error("Supabase error fetching scenes list:", error);
   throw error;
 }

 let scenes = (data || []).map(s => ({
   id: s.id,
   scene_slug: s.scene_slug,
   title_for_admin: s.title_for_admin,
   updated_at: s.updated_at,
   tags: s.tags || [],
   choices_count: Array.isArray(s.choices) && s.choices.length > 0 && typeof s.choices[0].count === 'number'
                  ? s.choices[0].count
                  : 0,
 })) as SceneListEntry[];

 if (orderBy === 'choices_count') {
   scenes.sort((a, b) => {
     const countA = a.choices_count || 0;
     const countB = b.choices_count || 0;
     return orderDirection === 'asc' ? countA - countB : countB - countA;
   });
 }

 return { scenes, totalCount: count || 0 };
};

/**
* Ruft eine einzelne Szene anhand ihrer ID ab, inklusive aller zugehörigen Choices.
* @param sceneId - Die ID der abzurufenden Szene.
* @returns Die Szene oder null, falls nicht gefunden.
*/
export const fetchSceneById = async (sceneId: string): Promise<SceneData | null> => {
 const { data, error } = await supabase
   .from('scenes')
   .select('*, choices(*)')
   .eq('id', sceneId)
   .single();

 if (error) {
   console.error("Supabase error fetching scene by id:", error);
   if (error.code === 'PGRST116') {
       return null;
   }
   throw error;
 }

 if (data && data.choices) {
   data.choices.sort((a: ChoiceData, b: ChoiceData) => (a.order_in_scene ?? Infinity) - (b.order_in_scene ?? Infinity));
 }
 if (data && data.tags === null) {
   data.tags = [];
 }

 return data as SceneData | null;
};

/**
* Speichert eine Szene (erstellt eine neue oder aktualisiert eine bestehende).
* Beinhaltet das Management der zugehörigen Choices.
* @param scenePayload - Die Daten der zu speichernden Szene. 'id' ist optional (für neue Szenen).
* @param choicesPayload - Ein Array der Choice-Daten, die zur Szene gehören.
* @returns Die gespeicherte Szene.
*/
export const saveScene = async (
 scenePayload: Partial<SceneData>,
 choicesPayload: ChoiceData[]
): Promise<SceneData> => {
 const { id, choices, created_at, updated_at, choices_count, ...sceneDetailsToSave } = scenePayload as SceneData;

 let savedSceneResult: SceneData;

 if (id) { // Bestehende Szene aktualisieren
   const { data: updatedSceneData, error: updateSceneError } = await supabase
     .from('scenes')
     .update(sceneDetailsToSave)
     .eq('id', id)
     .select()
     .single();

   if (updateSceneError) {
     console.error("Supabase error updating scene:", updateSceneError);
     throw updateSceneError;
   }
   savedSceneResult = updatedSceneData as SceneData;

   const currentChoiceIdsInForm = choicesPayload.map(c => c.id).filter(Boolean) as string[];
   const { data: originalChoicesData, error: fetchOriginalChoicesError } = await supabase
       .from('choices')
       .select('id')
       .eq('scene_id', savedSceneResult.id!);

   if (fetchOriginalChoicesError) {
       console.warn("Could not fetch original choices for deletion check", fetchOriginalChoicesError);
   }
   const originalChoiceIds = (originalChoicesData || []).map(c => c.id);
   const choicesToDelete = originalChoiceIds.filter(origId => !currentChoiceIdsInForm.includes(origId));

   if (choicesToDelete.length > 0) {
     const { error: deleteError } = await supabase.from('choices').delete().in('id', choicesToDelete);
     if (deleteError) {
       console.warn("Supabase error deleting choices:", deleteError.message);
     }
   }

   const choicesForUpsert = choicesPayload.map((choice, index) => {
       const { temp_id, created_at: c_created_at, updated_at: c_updated_at, ...choiceFields } = choice;
       return {
           ...choiceFields,
           scene_id: savedSceneResult.id!,
           order_in_scene: index
       };
   });

   if (choicesForUpsert.length > 0) {
       const { data: upsertedChoicesData, error: upsertChoicesError } = await supabase
       .from('choices')
       .upsert(choicesForUpsert, { onConflict: 'id', ignoreDuplicates: false })
       .select();

       if (upsertChoicesError) {
           console.error("Supabase error upserting choices:", upsertChoicesError);
           throw upsertChoicesError;
       }
       savedSceneResult.choices = (upsertedChoicesData || []).sort((a,b) => (a.order_in_scene ?? 0) - (b.order_in_scene ?? 0));
   } else {
       savedSceneResult.choices = [];
   }

 } else { // Neue Szene erstellen
   const { data: newSceneData, error: insertSceneError } = await supabase
     .from('scenes')
     .insert(sceneDetailsToSave)
     .select()
     .single();

   if (insertSceneError) {
     console.error("Supabase error inserting scene:", insertSceneError);
     throw insertSceneError;
   }
   savedSceneResult = newSceneData as SceneData;

   if (choicesPayload.length > 0) {
     const choicesToInsert = choicesPayload.map((choice, index) => {
       const { id: choiceId, temp_id, created_at: c_created_at, updated_at: c_updated_at, ...choiceFieldsNoId } = choice;
       return {
           ...choiceFieldsNoId,
           scene_id: savedSceneResult.id!,
           order_in_scene: index
       };
     });

     const { data: insertedChoicesData, error: choicesError } = await supabase
       .from('choices')
       .insert(choicesToInsert)
       .select();

     if (choicesError) {
       console.error("Supabase error inserting choices for new scene:", choicesError);
       throw choicesError;
     }
     savedSceneResult.choices = (insertedChoicesData || []).sort((a,b) => (a.order_in_scene ?? 0) - (b.order_in_scene ?? 0));
   } else {
     savedSceneResult.choices = [];
   }
 }
 if (savedSceneResult && savedSceneResult.tags === null) {
   savedSceneResult.tags = [];
 }
 return savedSceneResult;
};

/**
* Löscht eine Szene anhand ihrer ID.
* Zugehörige Choices werden durch 'ON DELETE CASCADE' in der Datenbank automatisch mitgelöscht.
* @param sceneId - Die ID der zu löschenden Szene.
*/
export const deleteScene = async (sceneId: string): Promise<void> => {
 const { error } = await supabase
   .from('scenes')
   .delete()
   .eq('id', sceneId);

 if (error) {
   console.error("Supabase error deleting scene:", error);
   throw error;
 }
};

// File: src/components/Shared/KeyValueEditor.tsx
import React, { useState, useEffect, useCallback } from 'react';
import { Box, Typography, TextField, IconButton, Button, Stack, FormHelperText, Tooltip } from '@mui/material';
import AddCircleOutlineIcon from '@mui/icons-material/AddCircleOutline';
import RemoveCircleOutlineIcon from '@mui/icons-material/RemoveCircleOutline';
import { v4 as uuidv4 } from 'uuid';

interface KeyValuePair {
 id: string;
 keyField: string;
 valueField: string;
}

interface KeyValueEditorProps {
 label: string;
 values: Record<string, any>;
 onChange: (newValues: Record<string, any>) => void;
 disabled?: boolean;
 keyHelperText?: string;
 valueHelperText?: string;
 keyLabel?: string;
 valueLabel?: string;
 addButtonText?: string;
 containerSx?: object;
}

const KeyValueEditor: React.FC<KeyValueEditorProps> = ({
 label,
 values,
 onChange,
 disabled = false,
 keyHelperText = 'Eindeutiger Name des Flags/Schlüssels.',
 valueHelperText = 'Wert: true, false, Zahl (z.B. 10), Text (z.B. "gefunden"), "+5" (addiert 5), "-2" (subtrahiert 2).',
 keyLabel = "Schlüssel / Flag-Name",
 valueLabel = "Wert",
 addButtonText,
 containerSx = {},
}) => {
 const [pairs, setPairs] = useState<KeyValuePair[]>([]);

 useEffect(() => {
   const initialPairs = Object.entries(values || {}).map(([key, value]) => ({
     id: uuidv4(),
     keyField: key,
     valueField: String(value),
   }));
   setPairs(initialPairs);
 }, [values]);

 const triggerOnChange = useCallback((currentPairs: KeyValuePair[]) => {
   const newValuesObject: Record<string, any> = {};
   currentPairs.forEach(p => {
     const key = p.keyField.trim();
     if (key !== '') {
       const valStr = p.valueField.trim();
       if (valStr.toLowerCase() === 'true') {
         newValuesObject[key] = true;
       } else if (valStr.toLowerCase() === 'false') {
         newValuesObject[key] = false;
       } else if (!isNaN(Number(valStr)) && valStr !== '' && !valStr.startsWith('+') && !valStr.startsWith('-')) {
         newValuesObject[key] = Number(valStr);
       } else {
         newValuesObject[key] = p.valueField;
       }
     }
   });
   onChange(newValuesObject);
 }, [onChange]);

 const handlePairChange = (index: number, field: 'keyField' | 'valueField', newValue: string) => {
   const newPairs = pairs.map((pair, i) =>
     i === index ? { ...pair, [field]: newValue } : pair
   );
   setPairs(newPairs);
 };

 const addPair = () => {
   const newPairs = [...pairs, { id: uuidv4(), keyField: '', valueField: '' }];
   setPairs(newPairs);
 };

 const removePair = (idToRemove: string) => {
   const newPairs = pairs.filter((pair) => pair.id !== idToRemove);
   setPairs(newPairs);
   triggerOnChange(newPairs);
 };

 const handleBlur = () => {
   triggerOnChange(pairs);
 };

 return (
   <Box sx={{ mb: 2, p: 2, border: '1px solid', borderColor: 'divider', borderRadius: '8px', ...containerSx }}>
     <Typography variant="subtitle1" gutterBottom color="text.primary" sx={{ fontWeight: 'medium', mb: 1.5 }}>
       {label}
     </Typography>
     {pairs.length === 0 && !disabled && (
       <Typography variant="body2" color="text.disabled" sx={{ fontStyle: 'italic', textAlign: 'center', mb: 1.5 }}>
         Keine Einträge vorhanden.
       </Typography>
     )}
     {pairs.map((pair, index) => (
       <Stack direction="row" spacing={1.5} key={pair.id} sx={{ mb: 2, alignItems: 'flex-start' }}>
         <TextField
           label={keyLabel}
           value={pair.keyField}
           onChange={(e) => handlePairChange(index, 'keyField', e.target.value)}
           onBlur={handleBlur}
           size="small"
           sx={{ flexGrow: 3 }}
           disabled={disabled}
           helperText={index === 0 ? keyHelperText : ''}
           variant="outlined"
           InputLabelProps={{ shrink: true }}
         />
         <TextField
           label={valueLabel}
           value={pair.valueField}
           onChange={(e) => handlePairChange(index, 'valueField', e.target.value)}
           onBlur={handleBlur}
           size="small"
           sx={{ flexGrow: 2 }}
           disabled={disabled}
           helperText={index === 0 ? valueHelperText : ''}
           variant="outlined"
           InputLabelProps={{ shrink: true }}
         />
         <Tooltip title="Diesen Eintrag entfernen">
           <IconButton
             onClick={() => removePair(pair.id)}
             color="error"
             size="medium"
             disabled={disabled}
             aria-label={`Eintrag ${index + 1} entfernen`}
             sx={{ mt: '5px' }}
           >
             <RemoveCircleOutlineIcon />
           </IconButton>
         </Tooltip>
       </Stack>
     ))}
     <Button
       startIcon={<AddCircleOutlineIcon />}
       onClick={addPair}
       size="small"
       variant="outlined"
       color="primary"
       disabled={disabled}
       sx={{ mt: pairs.length > 0 ? 0 : 1, textTransform: 'none', fontWeight: 'medium' }}
     >
       {addButtonText || `Eintrag hinzufügen`}
     </Button>
   </Box>
 );
};

export default KeyValueEditor;

       id: sessionDetails.current_scene.id,
       slug: sessionDetails.current_scene.scene_slug,
       title: sessionDetails.current_scene.title_for_admin,
       text: sessionDetails.current_scene.text_for_player || "Die Mysterien dieser Szene sind noch verborgen...",
       imageUrl: sessionDetails.current_scene.image_url,
       musicUrl: sessionDetails.current_scene.music_url,
       choices: (sessionDetails.current_scene.choices || [])
           .sort((a: any, b: any) => (a.order_in_scene ?? 0) - (b.order_in_scene ?? 0))
           .filter((c: any) => c.player_facing_text)
           .map((c: any) => ({ id: c.id, text: c.player_facing_text }))
     });

     setPlayerState({
       flags: sessionDetails.player_flags || {},
       stats: sessionDetails.player_stats || {},
       inventory: sessionDetails.player_inventory || [],
     });
     setShowScene(true);
   } catch (err: any) {
     console.error("Error fetching initial/current state:", err);
     showToast(err.message || 'Fehler beim Laden des Spiels. Bitte versuche, ein neues Spiel zu starten.', 'error');
     navigate('/play/start');
   } finally {
     setIsLoading(false);
   }
 }, [sessionId, navigate, showToast]);

 useEffect(() => {
   fetchInitialOrCurrentState();
 }, [fetchInitialOrCurrentState]);

 const handleChoiceClick = async (choiceId: string) => {
   if (!sessionId) return;
   setIsChoosing(true);
   setShowScene(false);

   try {
     const { data, error: functionError } = await supabase.functions.invoke<NextSceneResponse>(
       'generate-next-scene',
       {
         body: {
           player_session_id: sessionId,
           chosen_option_id: choiceId,
         },
       }
     );

     if (functionError) {
       console.error("Function Error (generate-next-scene):", functionError);
       throw new Error(functionError.message || 'Fehler beim Generieren der nächsten Szene.');
     }

     if (data && data.new_scene_id) {
       setCurrentScene({
         id: data.new_scene_id,
         slug: data.scene_slug,
         title: data.scene_title,
         text: data.scene_text,
         imageUrl: data.image_url,
         musicUrl: data.music_url,
         choices: data.choices.filter(c => c.text),
       });
       setPlayerState(data.updated_player_state);
       showToast("Die Geschichte entwickelt sich...", "info");
       setShowScene(true);
       if (audioRef.current && data.music_url) {
           if(isMusicPlaying) audioRef.current.play().catch(e => console.warn("Audio autoplay prevented:", e));
       }
     } else {
       showToast('Keine gültigen Daten von der nächsten Szene erhalten. Das Orakel schweigt.', 'warning');
       setShowScene(true);
     }
   } catch (err: any) {
     console.error("Error handling choice:", err);
     showToast(err.message || 'Fehler beim Verarbeiten deiner Wahl.', 'error');
     setShowScene(true);
   } finally {
     setIsChoosing(false);
   }
 };

 const toggleMusic = () => {
   if (audioRef.current) {
     if (isMusicPlaying) {
       audioRef.current.pause();
     } else {
       audioRef.current.play().catch(e => console.warn("Audio play prevented:", e));
     }
     setIsMusicPlaying(!isMusicPlaying);
   }
 };

 useEffect(() => {
   if (audioRef.current) {
     if (isMusicPlaying && currentScene?.musicUrl) {
       audioRef.current.play().catch(e => console.warn("Audio autoplay prevented on load:", e));
     } else {
       audioRef.current.pause();
     }
   }
 }, [isMusicPlaying, currentScene?.musicUrl]);

 if (isLoading) {
   return (
     <Box sx={{ display: 'flex', flexDirection: 'column', justifyContent: 'center', alignItems: 'center', height: '100vh', p: 2, background: theme.palette.grey[900], color: 'white' }}>
       <CircularProgress size={60} color="secondary"/>
       <Typography variant="h6" sx={{ mt: 3, fontFamily: '"Quintessential", serif' }}>Das magische Buch öffnet seine Seiten...</Typography>
     </Box>
   );
 }

 if (!currentScene) {
   return (
     <Container maxWidth="md" sx={{ py: 4, textAlign: 'center', background: theme.palette.grey[900], minHeight: '100vh', color: 'white' }}>
       <Paper elevation={3} sx={{ p: 3, backgroundColor: 'rgba(0,0,0,0.7)', backdropFilter: 'blur(5px)' }}>
           <Typography variant="h5" gutterBottom sx={{fontFamily: '"Cinzel", serif'}}>Ein unerwartetes Ende?</Typography>
           <Alert severity="warning" sx={{mb: 2}}>Die aktuelle Szene konnte nicht geladen werden. Möglicherweise ist das Abenteuer hier zu Ende oder ein Fehler ist aufgetreten.</Alert>
           <Button variant="contained" color="secondary" onClick={() => navigate('/play/start')} sx={{mr:1}}>
           Neues Spiel starten
           </Button>
           <Button variant="outlined" color="primary" onClick={fetchInitialOrCurrentState} disabled={isLoading}>
           Erneut versuchen
           </Button>
       </Paper>
     </Container>
   );
 }

 return (
   <Container
       maxWidth="lg"
       sx={{
           py: {xs: 2, sm: 3, md: 4},
           minHeight: '100vh',
           display: 'flex',
           flexDirection: 'column',
           background: 'url("https://www.transparenttextures.com/patterns/old-paper.png"), linear-gradient(to bottom, #614A3A, #3E2E22)',
           backgroundBlendMode: 'overlay, normal',
           color: '#EAE0D5',
           fontFamily: '"Roboto Slab", serif',
       }}
   >
     {currentScene.musicUrl && (
       <audio ref={audioRef} src={currentScene.musicUrl} loop onError={(e) => console.warn("Error loading audio:", e.currentTarget.error)}>
           Dein Browser unterstützt das Audio-Element nicht.
       </audio>
     )}
     <Grow in={showScene} timeout={700}>
       <Paper
           elevation={10}
           sx={{
           p: { xs: 2, sm: 3, md: 4 },
           mb: 3,
           backgroundColor: 'rgba(40, 30, 25, 0.92)',
           backdropFilter: 'blur(4px)',
           border: '3px double #B08D57',
           borderRadius: '8px',
           boxShadow: '0 12px 25px rgba(0,0,0,0.6)',
           color: '#EAE0D5',
           }}
       >
           {currentScene.imageUrl ? (
           <CardMedia
               component="img"
               image={currentScene.imageUrl}
               alt={currentScene.title || 'Szenenbild'}
               sx={{
               width: '100%',
               maxHeight: {xs: '200px', sm: '250px', md: '350px'},
               objectFit: 'cover',
               mb: 2.5,
               borderRadius: '4px',
               border: '2px solid #8C6B42',
               boxShadow: 'inset 0 0 10px rgba(0,0,0,0.5)',
               }}
               onError={(e: React.SyntheticEvent<HTMLImageElement, Event>) => {
                   (e.target as HTMLImageElement).style.display = 'none';
               }}
           />
           ) : (
               <Box sx={{display: 'flex', justifyContent:'center', alignItems:'center', height: {xs: 100, sm:150}, backgroundColor: 'rgba(0,0,0,0.2)', mb: 2.5, borderRadius: '4px', border: '2px dashed #8C6B42'}}>
                   <ImageIcon sx={{fontSize: 50, color: '#8C6B42'}}/>
               </Box>
           )}

           <Typography variant="h4" component="h1" gutterBottom sx={{ fontFamily: '"Cinzel Decorative", cursive', fontWeight: 'bold', color: '#FFD700', textAlign: 'center', textShadow: '1px 1px 3px rgba(0,0,0,0.7)' }}>
           {currentScene.title || 'Ein unbekannter Ort'}
           </Typography>
           <Divider sx={{ my: 2, borderColor: 'rgba(176, 141, 87, 0.5)' }} />
           <Typography variant="body1" component="div" sx={{ whiteSpace: 'pre-wrap', mb: 3, fontSize: '1.15rem', lineHeight: 1.8, fontFamily: '"EB Garamond", serif', textAlign: 'justify', hyphens: 'auto' }}>
           {currentScene.text}
           </Typography>
       </Paper>
     </Grow>
     <Collapse in={showScene && !isChoosing}>
       <Paper
           elevation={6}
           sx={{
               p: { xs: 1.5, sm: 2, md: 3 },
               backgroundColor: 'rgba(60, 45, 38, 0.88)',
               backdropFilter: 'blur(3px)',
               border: '2px solid #A17A48',
               borderRadius: '8px',
               mb: 3,
           }}
       >
           <Typography variant="h6" sx={{ mb: 2, fontFamily: '"MedievalSharp", cursive', color: '#FFDEAD', textAlign:'center' }}>Deine nächste Handlung?</Typography>
           {isChoosing && <Box sx={{display: 'flex', justifyContent: 'center', my:3}}><CircularProgress color="secondary"/></Box>}

           {!isChoosing && currentScene.choices && currentScene.choices.length > 0 && (
               <List disablePadding>
               {currentScene.choices.map((choice, index) => (
                   <Slide direction="up" in={showScene} timeout={500 + index * 150} key={choice.id}>
                   <ListItem disablePadding sx={{mb: 1.5}}>
                       <ListItemButton
                       onClick={() => handleChoiceClick(choice.id)}
                       disabled={isChoosing}
                       sx={{
                           py: 1.8,
                           px: 2.5,
                           transition: 'background-color 0.3s, transform 0.2s',
                           fontFamily: '"MedievalSharp", cursive',
                           fontSize: '1.25rem',
                           color: '#EAE0D5',
                           border: '2px solid rgba(176, 141, 87, 0.6)',
                           borderRadius: '6px',
                           textAlign: 'center',
                           justifyContent: 'center',
                           '&:hover': {
                               backgroundColor: 'rgba(255, 222, 173, 0.15)',
                               color: '#FFD700',
                               transform: 'scale(1.02)',
                               borderColor: '#FFD700',
                           },
                       }}
                       >
                       {choice.text}
                       </ListItemButton>
                   </ListItem>
                   </Slide>
               ))}
               </List>
           )}

           {!isChoosing && (!currentScene.choices || currentScene.choices.length === 0) && (
               <Typography sx={{my: 2, fontStyle: 'italic', textAlign: 'center', fontFamily: '"Quintessential", serif', color: 'rgba(234, 224, 213, 0.7)'}}>
                   Das Buch scheint an dieser Stelle keine weiteren Pfade zu offenbaren. Das Abenteuer endet hier... vorerst.
               </Typography>
           )}
       </Paper>
     </Collapse>
     {playerState && (
       <Collapse in={showScene}>
       <Paper elevation={2} sx={{ p: 2, mt: 'auto', mb:2, backgroundColor: 'rgba(0,0,0,0.5)', fontSize: '0.85rem', color: 'rgba(255,255,255,0.75)', borderRadius: '8px' }}>
         <Typography variant="overline" display="block" sx={{color: '#FFDEAD', mb: 0.5}}>Status des Abenteurers:</Typography>
         <Typography variant="caption" sx={{fontFamily: 'monospace', display: 'block', whiteSpace: 'pre-wrap', wordBreak: 'break-all', maxHeight: '100px', overflowY: 'auto'}}>
           Flags: {Object.keys(playerState.flags).length > 0 ? JSON.stringify(playerState.flags, null, 2) : 'Keine aktiven Flags'}
         </Typography>
       </Paper>
       </Collapse>
     )}
     <Stack direction="row" spacing={2} justifyContent="center" sx={{ mt: 'auto', pt: 2, pb:1 }}>
           {currentScene.musicUrl && (
                <Tooltip title={isMusicPlaying ? "Musik stummschalten" : "Musik abspielen"}>
                   <IconButton onClick={toggleMusic} sx={{color: 'rgba(255,222,173,0.8)', '&:hover': {color: '#FFDEAD'}}}>
                       {isMusicPlaying ? <VolumeUpIcon /> : <VolumeOffIcon />}
                   </IconButton>
               </Tooltip>
           )}
           <Button
               variant="outlined"
               startIcon={<HomeIcon />}
               onClick={() => navigate('/')}
               sx={{borderColor: 'rgba(255,222,173,0.5)', color: 'rgba(255,222,173,0.8)', '&:hover': {borderColor: '#FFDEAD', color: '#FFDEAD', backgroundColor: 'rgba(255,222,173,0.1)'}}}
           >
               Hauptmenü
           </Button>
           <Button
               variant="contained"
               color="secondary"
               startIcon={<ReplayIcon />}
               onClick={() => navigate('/play/start')}
                sx={{'&:hover': {backgroundColor: 'secondary.dark'}}}
           >
               Neues Spiel
           </Button>
       </Stack>
   </Container>
 );
};
